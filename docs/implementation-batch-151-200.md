# Implementation batch 151-200

Reviewed on 2026-07-18. This ledger is the execution overlay for backlog items 151-200 in [recommendation.md](../recommendation.md) and [ideas-top-300.md](ideas-top-300.md). It separates active resident behavior from bounded contracts and keeps SQLCipher, native integrations, real Wayland sessions, mobile apps, live IPC dispatch, and sync transport explicit.

## Status vocabulary

| Status | Meaning |
|---|---|
| **Runtime** | Exercised by the current resident binary, CLI, store, packaging, or CI path. |
| **Foundation** | Implemented and tested as a reusable contract or algorithm, but not connected to its final runtime. |
| **Adapted** | The proposal was narrowed to preserve safety, privacy, or an honest product boundary. |
| **Native required** | Completion depends on real per-OS APIs, compositor sessions, mobile surfaces, or release infrastructure. |
| **External required** | Completion requires time-bound human or operational evidence that cannot be generated by a unit test. |
| **Rejected** | The mechanism conflicts with a security or correctness constraint; the replacement is recorded. |

## Acceptance gate

This batch is accepted only after formatting, strict all-target/all-feature clippy, locked workspace tests, no-default root tests, WASM compilation, Linux golden comparison without update mode, documentation contracts/local links, workflow syntax review, and `git diff --check` pass. External/native rows remain gates even after their deterministic contracts are green.

The final local acceptance run completed on macOS, including strict workspace clippy/tests, no-default tests, WASM compilation, release performance budgets, and a byte-identical 16-image comparison on Linux `amd64` Mesa/Vulkan. GitHub Actions independently repeats the macOS/Windows/Linux matrix for the pushed commit; its live result belongs to that commit's checks and is not pre-claimed here. Credentialed signing, real compositor reports, hosted demo deployment, two-week dogfood, and live network/integration systems remain intentionally open.
